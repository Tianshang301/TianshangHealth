# TianshangHealth(天殇·悦康)

> **Приватний, повністю офлайн Android-додаток для здоров'я.**
> Поєднує відстеження менструального циклу, фітнес, сон, харчування та підрахунок кроків в одній зашифрованій локальній базі даних — з багатовимірним аналізом на основі локального ШІ.

---

## Ключові особливості

- **Нульові мережеві дозволи** — Всі дані залишаються на вашому пристрої. Без хмар, без аналітики, без Firebase.
- **Шифрування військового рівня** — База даних SQLCipher AES-256 захищена апаратними ключами Android Keystore.
- **Модульна архітектура** — 12 модулів Gradle за шаблоном MVVM + Repository + Hilt + Compose.
- **Локальний ШІ-інференс** — Моделі TensorFlow Lite покращують прогнозування циклу та аналіз настрою, не залишаючи пристрій.
- **21 мова** — Включаючи підтримку RTL для арабської; локалізовано з першого дня.
- **Адаптивний інтерфейс за статтю** — Нижня навігація та функції адаптуються залежно від вибраної статі.

## Функції

| Сфера | Можливості |
|-------|------------|
| **Відстеження циклу** | Перегляд календаря, двигун прогнозування циклу (IQR-фільтрація + експоненціальне затухання), журнал симптомів, нагадування |
| **Кроки** | Фонова служба апаратного крокоміра + резервний акселерометр, синхронізація WorkManager, аналіз кроків за фазами циклу |
| **Фітнес** | 17 типів вправ, MET-калькулятор калорій, рекомендації щодо тренувань з урахуванням фази циклу |
| **Сон** | Ручне введення сну, оцінка якості, трендовий графік Canvas |
| **Харчування** | Журнал харчування (сніданок/обід/вечеря/перекус), відстеження макроелементів, споживання води |
| **Аналіз** | Багатовимірний аналітичний двигун, рекомендації щодо здоров'я, експорт медичних звітів, покращення прогнозів TFLite |
| **Безпека** | PIN-код (Argon2id) + біометричне блокування, зашифроване ZIP-резервування/відновлення, захист від знімків екрана на чутливих екранах |

## Архітектура

```
app/                    # Оболонка додатку: єдиний NavHost, тема, MainActivity
├── navigation/MainNavigation.kt        # Умовна нижня навігація + бічні маршрути
core/
├── common/             # Спільні компоненти UI, утиліти, всі рядкові ресурси (21 мова)
├── database/           # Сутності Room, DAO, репозиторії, міграції
└── security/           # Keystore, SQLCipher, Argon2id, біометрична автентифікація, блокування додатку
feature/
├── onboarding/         # Вибір статі при першому запуску
├── dashboard/          # Огляд дня, інсайти здоров'я
├── period/             # Відстеження циклу, прогнозування, нагадування, резервне копіювання, налаштування
├── steps/              # Служба крокоміра + синхронізація WorkManager
├── fitness/            # Записи тренувань + рекомендації з урахуванням фази циклу
├── sleep/              # Записи сну + трендовий графік
├── nutrition/          # Відстеження їжі та води
└── analysis/           # Багатовимірний аналіз + інференс ML TFLite
```

## Безпека та конфіденційність

- **Дані в стані спокою**: SQLCipher AES-256 шифрує всю базу даних `tianshang_health.db`. Пароль бази даних — випадковий рядок із 32 символів, зашифрований Android Keystore, який ніколи не зберігається у відкритому вигляді.
- **Автентифікація**: PIN-коди хешуються за допомогою Argon2id. BiometricPrompt (`BIOMETRIC_STRONG`) доступний як резервний варіант.
- **Мережа**: `AndroidManifest.xml` **не містить дозволу `INTERNET`**. Без Firebase, без аналітики, без звітів про збої.
- **Блокування додатку**: При переході у фон активується настроювана затримка (негайно / 30с / 1хв / 5хв). Екран блокування перекриває весь додаток із `zIndex(10f)`.
- **Резервне копіювання**: Зашифрований ZIP-експорт/відновлення з паролем, наданим користувачем. Також підтримується експорт CSV.
- **Посилення захисту**: `FLAG_SECURE` на чутливих екранах; перевірка `Debug.isDebuggerConnected()` під час запуску.

 ## Початок роботи

### Передумови

- JDK 17
- Android Studio (рекомендується Ladybug або новіша версія)
- Android SDK із встановленим API 35

### Збірка Debug APK

```bash
./gradlew :app:assembleDebug
```

APK буде розташовано за адресою:
```
app/build/outputs/apk/debug/app-debug.apk
```

### Встановлення на пристрій

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### Збірка Release APK

Release-збірки вимагають конфігурації підпису в `local.properties`:

```properties
RELEASE_STORE_FILE=path/to/keystore.jks
RELEASE_STORE_PASSWORD=your-store-password
RELEASE_KEY_ALIAS=your-alias
RELEASE_KEY_PASSWORD=your-key-password
```

Потім запустіть:

```bash
./gradlew :app:assembleRelease
```

Підписаний APK матиме назву:
```
app/build/outputs/apk/release/TianshangHealth-v1.0.0.apk
```

### Передрелізна перевірка безпеки

```bash
python check_release_security.py
```
