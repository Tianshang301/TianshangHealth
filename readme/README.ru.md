# TianshangHealth(天殇·悦康)

> **Приватный, полностью автономный Android-компаньон для здоровья.**
> Объединяет отслеживание менструального цикла, фитнес, сон, питание и подсчёт шагов в одной зашифрованной локальной базе данных — с многомерной аналитикой на основе локального ИИ.

---

## Ключевые особенности

- **Нулевые сетевые разрешения** — Все данные остаются на вашем устройстве. Никаких облаков, аналитики, Firebase.
- **Военный уровень шифрования** — База данных SQLCipher AES-256 защищена аппаратными ключами Android Keystore.
- **Модульная архитектура** — 12 Gradle-модулей по шаблону MVVM + Repository + Hilt + Compose.
- **Локальное ИИ-инференс** — Модели TensorFlow Lite улучшают прогнозирование менструаций и анализ настроения, не покидая устройство.
- **21 язык** — Включая RTL-поддержку арабского языка; локализован с первого дня.
- **Адаптивный интерфейс по полу** — Нижняя навигация и функции адаптируются в зависимости от выбранного пользователем пола.

## Функции

| Область | Возможности |
|--------|-------------|
| **Отслеживание цикла** | Календарь, движок прогнозирования циклов (IQR-фильтрация + экспоненциальное затухание), журнал симптомов, напоминания |
| **Шаги** | Фоновый сервис аппаратного шагомера + резервный акселерометр, синхронизация WorkManager, анализ шагов по фазам цикла |
| **Фитнес** | 17 типов упражнений, расчёт калорий по MET, рекомендации по тренировкам с учётом фазы цикла |
| **Сон** | Ручной ввод сна, оценка качества, трендовый график Canvas |
| **Питание** | Журнал питания (завтрак/обед/ужин/перекус), отслеживание макронутриентов, потребление воды |
| **Анализ** | Многомерный аналитический движок, рекомендации по здоровью, экспорт медицинских отчётов, улучшение прогнозов TFLite |
| **Безопасность** | PIN-код (Argon2id) + биометрическая блокировка, зашифрованное ZIP-резервирование/восстановление, защита от скриншотов на конфиденциальных экранах |

## Безопасность и конфиденциальность

- **Шифрование данных в состоянии покоя**: Вся база данных `tianshang_health.db` зашифрована с помощью SQLCipher AES-256. Пароль базы данных — случайная 32-символьная строка, зашифрованная Android Keystore и никогда не покидающая Keystore.
- **Аутентификация**: PIN-коды хешируются с помощью Argon2id. BiometricPrompt (`BIOMETRIC_STRONG`) доступен в качестве резервного варианта.
- **Сетевая изоляция**: В `AndroidManifest.xml` **нет разрешения `INTERNET`**. Нет Firebase, нет аналитики, нет отчётов о сбоях.
- **Блокировка приложения**: При переходе приложения в фоновый режим срабатывает блокировка с настраиваемой задержкой (немедленно / 30с / 1мин / 5мин). `AppLockScreen` перекрывает всё приложение с `zIndex(10f)`.
- **Резервное копирование и экспорт**: Поддерживается зашифрованное ZIP-резервирование/восстановление с паролем пользователя. Также поддерживается экспорт в CSV.
- **Усиление защиты**: `FLAG_SECURE` на конфиденциальных экранах; проверка `Debug.isDebuggerConnected()` при запуске.

## Быстрый старт

### Требования

- JDK 17
- Android Studio (рекомендуется Ladybug или новее)
- Android SDK с установленным API 35

### Сборка Debug APK

```bash
./gradlew :app:assembleDebug
```

APK будет расположен по пути:
```
app/build/outputs/apk/debug/app-debug.apk
```

### Установка на устройство

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### Сборка Release APK

Для сборки Release требуется настройка подписи в `local.properties`:

```properties
RELEASE_STORE_FILE=path/to/keystore.jks
RELEASE_STORE_PASSWORD=your-store-password
RELEASE_KEY_ALIAS=your-alias
RELEASE_KEY_PASSWORD=your-key-password
```

Затем выполните:

```bash
./gradlew :app:assembleRelease
```

Подписанный APK будет назван:
```
app/build/outputs/apk/release/TianshangHealth-v1.0.0.apk
```

### Предварительная проверка безопасности

```bash
python check_release_security.py
```
